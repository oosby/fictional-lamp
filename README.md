# Fictional Lamp
This shite c++ is an attempt to create a rudimentary api for a web app to use

## Server

### compile and run the server
- open a fresh tab in your terminal
- `cd server`
- `g++ -o main main.cpp`
- `./main`

## client

### run the client
- open a fresh tab in your terminal
- `cd client`
- `python3 -m http.server 8000`
